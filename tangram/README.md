# 🔷 Tangram

Jogo educativo de Tangram com peças interativas, criado com **PixiJS** + **TypeScript** + **Vite**.

## Funcionalidades

- 🎮 **Tela inicial** com seleção de desafios por dificuldade (FÁCIL, DIFÍCIL, PERSONALIZADO)
- ⏱️ **Cronômetro** com contagem regressiva 3-2-1 antes de cada desafio
- 🏆 **Ranking online** em tempo real via Supabase (top 3 por desafio)
- 🎨 **Modo criador** — alunos criam novas formas que ficam salvas no banco online
- 🔷 **Rotação suave** das peças com controle visual
- 🎉 **Tela de vitória** animada

## Tecnologias

- [PixiJS v8](https://pixijs.com/) — renderização WebGL
- [Supabase](https://supabase.com/) — banco de dados online e leaderboard em tempo real
- [Vite](https://vitejs.dev/) + TypeScript — build tool
