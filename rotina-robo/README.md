# 🤖 A Rotina do Robô

Jogo educativo de lógica e ordenação voltado para crianças nos anos iniciais do Ensino Fundamental.

![Thumbnail](thumb.png)

## 🚀 Funcionalidades
- **48 Fases**: Desafios divididos em 12 mundos temáticos.
- **Login Automático**: Integração via SSO com o Portal BrincaBytes.
- **Design Moderno**: Interface em Glassmorphism, otimizada para dispositivos móveis e desktop.
- **Sistema de Ranking**: Acompanhamento de pontuação e estrelas.
- **Modais Customizados**: Experiência premium sem alertas padrão de navegador.

## 🛠️ Tecnologias
- HTML5 / CSS3 (Vanilla)
- JavaScript (Vanilla)
- BrincaBytes SSO Handshake Implementation

## 📖 Como Jogar
Basta abrir o arquivo `index.html` em qualquer navegador moderno. Arraste os blocos para a ordem correta da rotina e clique em "Verificar".
