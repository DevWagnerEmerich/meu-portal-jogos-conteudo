Tutorial dos Modos de Jogo

Modo Clássico

Gire a roleta, escolha letras e acumule pontos para adivinhar a palavra ou frase!

Como Jogar:

  • Clique em "Girar" na roleta ou no logo central.
  • Se cair em um valor, escolha uma letra no alfabeto. Cada letra correta multiplica o valor sorteado.
  • Se cair em "Passou a Vez", o próximo jogador continua.
  • Se cair em "Perdeu Tudo", você perde os pontos acumulados na rodada e passa a vez.
  • Use o botão ⏱️ para um cronômetro de 10 segundos caso precise de tempo para pensar.
  • Após revelar pelo menos duas letras, você pode usar o botão ✔️ para tentar revelar a palavra inteira (se errar, perde a vez).
  • Use o botão ❌ para passar a vez a qualquer momento antes de girar ou após acertar uma letra.
  • O jogador que completar a palavra ganha os pontos acumulados.

Modo Resposta Rápida

Acelere o jogo com este modo eletrizante! As regras são similares ao modo Clássico, mas com algumas reviravoltas:

  • Pontuação da Palavra: Cada palavra começa valendo 2000 pontos.
  • Decremento de Pontos: A cada letra revelada no painel, o valor da palavra diminui. Seja rápido para maximizar seus ganhos!
  • Adivinhe Primeiro: O jogador que conseguir adivinhar a palavra corretamente leva o valor atualizado da palavra.
  • "Perdeu Tudo" Diferente: Se cair em "Perdeu Tudo" na roleta, o jogador perde apenas os pontos acumulados com as letras *nesta rodada específica*, mas não afeta a pontuação principal da palavra que está em jogo.
  • Outras Regras:
    • Gire a roleta para obter valores ou ações.
    • Escolha letras para revelar no painel. Pontos da roleta são acumulados na rodada.
    • Use o botão ✔️ para tentar adivinhar a palavra. Se errar, perde a vez.
    • Use o botão ❌ para passar a vez.

O objetivo é adivinhar a palavra o mais rápido possível para garantir a maior quantidade de pontos da palavra!
